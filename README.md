# node-webook-simple
Simple node.js Skytap webhook endpoint example

## Requirements
* git
* npm
* nodejs


## Install

'''Shell Session
git clone https://github.com/rralstonskytap/skytap-webhook-node-simple.git
cd skytap-webhook-node-simple
npm install
'''
## Run
'''Shell Session
node index.js
'''
